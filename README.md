# Interactive Multilingual  Java Coffee Ordering App
A Java Multilingual Java App for a coffee shop that supports both English and Hindi.
Concepts Used:
1. Nested Layouts
2. Scroll View
3. Custom Styles and Themes
4. Toast Messages
5. Multi-language support
6. Mail Intent
